# Agent Tool Specification: send_to_admin

## Purpose

Describes the `send_to_admin` tool available to the AI agent. This specification is designed to be inserted into the system prompt to give the agent clear, structured instructions on when and how to use the tool.

## Tool: send_to_admin

### Name
`send_to_admin`

### Description
Отправляет заявку администраторам с резюме диалога и контактами пользователя. Вызывайте когда пользователь хочет оставить заявку, связаться с менеджером, или его вопрос требует вмешательства человека.

### Parameters

| Параметр | Тип | Обязательный | Описание |
|---|---|---|---|
| `summary` | string | Да | Резюме диалога и потребности пользователя. Опишите кратко: что хотел пользователь, какие вопросы задавал, какое решение ищет |
| `contacts` | string | Нет | Контактные данные пользователя: телефон, email, ссылка на соцсети. Если не предоставлены — напишите "Не предоставлены" |
| `username` | string | Да | Username пользователя в Telegram без символа @ |
| `telegram_link` | string | Да | Полная ссылка на профиль Telegram: https://t.me/username |

### When to Use

Вызывайте этот инструмент в следующих ситуациях:

1. Пользователь явно просит связаться с менеджером или администратором
2. Пользователь хочет оставить заявку, запрос или жалобу
3. Вопрос пользователя выходит за пределы вашей компетенции
4. Пользователь предоставил контактные данные и ждёт обратной связи
5. Диалог длится более 5 сообщений без разрешения вопроса

### When NOT to Use

Не вызывайте инструмент для:
- Простых приветствий и общих вопросов
- Справочной информации, которую вы можете дать сами
- Ответов на вопросы о возможностях бота
- Сообщений, на которые достаточно односложного ответа

### Behavior After Call

После успешной отправки заявки:
1. Сообщите пользователю: "Ваша заявка принята. Администраторы свяжутся с вами в ближайшее время."
2. Не продолжайте диалог — заявка передана, администратор возьмёт общение на себя

### Example Usage

```
User: "Мне нужна консультация по настройке рекламы, позвоните мне +998901234567"

Agent calls send_to_admin:
{
  "summary": "Пользователь запрашивает консультацию по настройке таргетированной рекламы. Интересуется настройкой кампаний в Instagram и Telegram.",
  "contacts": "Телефон: +998901234567",
  "username": "user_telegram",
  "telegram_link": "https://t.me/user_telegram"
}
```
